# Contributing

Thank you for contributing to Mechanical Realism Framework for Assetto Corsa (MRF).

## Pull Requests
- Fork the repository
- Create a feature branch
- Submit a PR to the dev branch (if used)

## Style
- Keep functions modular
- Comment mechanical logic clearly
- Avoid hardcoded car-specific tuning in core script
